
# Pixel Kombat Arena 🕹️

Ваша игровая платформа для Telegram.

## Как запустить?

### Способ 1: Прямая публикация (Самый быстрый)
1. Нажмите синюю кнопку **Publish** в верхней панели Firebase Studio.
2. Студия сама соберет проект и выдаст вам публичную ссылку.
3. Используйте эту ссылку в @BotFather.

### Способ 2: Через Vercel (Для полного контроля)
1. Скачайте проект (иконка стрелочки вниз в верхней панели). Если её нет — скопируйте код файлов вручную.
2. Создайте репозиторий на GitHub и загрузите туда файлы.
3. Подключите репозиторий к Vercel и добавьте API ключи.

## Основные функции
- **Arcade:** Тетрис, Соник и Бомбер.
- **PvP:** Режим "Экран + Джойстик". Один играет на ПК, второй управляет с телефона.
- **Local Referee:** Встроенная проверка честности, которая работает бесплатно.
- **Admin Panel:** Ручное подтверждение матчей и финансовых операций.

## Технологии
- **Next.js 15:** Быстрый фронтенд.
- **Firebase:** Бесплатная база данных и авторизация.
- **Gemini AI:** Умный анализ спорных моментов и уведомления.
